## 📂 Cara Instalasi
1. `git clone https://github.com/amdqeis/IMPAL`
2. `cd frontend`
3. `npm install`
4. `npm run dev`
5. `cd backend`
6. `python -m venv venv`
7. `source venv/bin/activate`  # Linux / Mac
8. `venv\Scripts\activate`     # Windows
9. `pip install -r requirements.txt`
10. `uvicorn main:app --reload`
